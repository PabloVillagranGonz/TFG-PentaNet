# 🎵 TFG - PentaNet: Sistema de Gestión para Centros Musicales

Este repositorio centraliza toda la documentación, recursos gráficos y enlaces a los diferentes componentes que integran el proyecto **PentaNet**, desarrollado como Trabajo de Fin de Grado (TFG).

PentaNet es una solución de software multiplataforma diseñada para optimizar la gestión de reservas de cabinas de estudio, control de asistencia, horarios y calificaciones en conservatorios y escuelas de música.

## 📦 Componentes del Proyecto

El ecosistema de PentaNet se divide en dos repositorios principales (Cliente y Servidor). Puede acceder al código fuente completo de cada uno a través de los siguientes enlaces:

| Componente | Descripción | Enlace al Repositorio |
| :--- | :--- | :--- |
| 📱 **PentaNet Client** | Aplicación multiplataforma desarrollada en **Flutter** para Alumnos, Profesores y personal de Secretaría. | [Ver Repositorio Frontend ↗](https://github.com/PabloVillagranGonz/PentaNetApp.git) |
| ⚙️ **PentaNet API** | Servidor backend desarrollado en **Java (Spring Boot)** con arquitectura REST y seguridad perimetral JWT. | [Ver Repositorio Backend ↗](https://github.com/PabloVillagranGonz/PentaNetApi.git) |
